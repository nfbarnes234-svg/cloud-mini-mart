# Build a Professional Cloud-Based Mini Mart POS System

You are a Senior Full-Stack Software Engineer and UI/UX Designer. Your task is to design and develop a modern, secure, scalable, cloud-based Point of Sale (POS) System for a mini mart.

## Technology Stack

Frontend:

* Next.js 15 (App Router)
* React 19
* TypeScript
* Tailwind CSS 4
* Shadcn/UI
* Lucide React Icons
* React Hook Form
* Zod Validation

Backend:

* Next.js API Routes
* Prisma ORM

Database:

* PostgreSQL

Authentication:

* JWT Authentication
* HTTP-only Cookies
* Role-Based Access Control

Cloud:

* Vercel (Frontend)
* Supabase PostgreSQL (Database)

State Management:

* React Context API

Charts:

* Recharts

Printing:

* Browser Receipt Printing
* 80mm Thermal Receipt Support

Barcode:

* USB Barcode Scanner Support
* Manual Barcode Entry
* Barcode Generation

Currency:

* Ghana Cedi (GHS)

Time Zone:

* Africa/Accra
